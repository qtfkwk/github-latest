# About

Get latest release tag for GitHub repositories

# Usage

~~~text
$ github-latest -V
github-latest 0.1.0
~~~

~~~text
$ github-latest -h
Get latest release tag for GitHub repositories

Usage: github-latest [REPOS]...

Arguments:
  [REPOS]...  One or more GitHub repositories (`qtfkwk/github-latest`)

Options:
  -h, --help     Print help
  -V, --version  Print version
~~~
